# 🚀 Portafolio Personal - Frontend Developer

Diseño de portafolio **limpio, minimalista y profesional** construido con tecnologías web estándar. Enfocado en la legibilidad y la experiencia de usuario (UX).

![Vista previa del proyecto](/img/pantallaPrevia.png)

## ✨ Características

- **Diseño Moderno:** Estética profesional con enfoque en tipografía y espacios en blanco.
- **Secciones clave:** Inicio, Blog (estilo editorial), Proyectos (con iconos animados) y Contacto.
- **Formulario Funcional:** Integración con **Formspree** para envío de correos vía AJAX (sin recarga de página).
- **Optimización SEO:** Metaetiquetas configuradas para mejor visibilidad en buscadores y redes sociales.
- **Totalmente Responsivo:** Adaptado para móviles, tablets y escritorio.

## 🛠️ Tecnologías utilizadas

- **HTML5** - Estructura semántica.
- **CSS3** - Flexbox, Grid y Animaciones personalizadas.
- **JavaScript (ES6+)** - Manipulación del DOM y Fetch API para el formulario.
- **Lucide Icons** - Iconografía moderna y ligera.
